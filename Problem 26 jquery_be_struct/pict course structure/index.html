PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics I - Foundation in calculus and linear algebra.
  • Engineering Physics - Basics of mechanics, optics, and thermodynamics.
  • Basic Electrical Engineering - Introduction to electric circuits and machines.
  • Engineering Graphics - Technical drawing and CAD tools.
  • Programming in C - Basic coding concepts and control structures.

Second Year (SE)

  • Data Structures - Stacks, queues, trees, and graphs.
  • Digital Logic Design - Boolean algebra and logic gates.
  • Computer Organization - CPU, memory, and I/O architecture.
  • Discrete Mathematics - Sets, relations, functions, and combinatorics.
  • Object-Oriented Programming - Classes, inheritance, and polymorphism.

Third Year (TE)

  • Database Management Systems - SQL, normalization, and transactions.
  • Operating Systems - Processes, memory, and file systems.
  • Computer Networks - TCP/IP, routing, and protocols.
  • Theory of Computation - Automata, grammars, and Turing machines.
  • Web Technologies - HTML, CSS, JS, and web frameworks.

Final Year (BE)

  • Artificial Intelligence - Search algorithms, ML basics.
  • Cyber Security - Threats, encryption, and firewalls.
  • Cloud Computing - IaaS, PaaS, SaaS, and virtualization.
  • Machine Learning - Supervised and unsupervised learning.
  • Project Work - Industry or research-based final year project.
